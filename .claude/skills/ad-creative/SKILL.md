### Ad Creative Guidelines

- **Image specs:** 1080x1080 (feed), 1080x1920 (Stories/Reels)
- **Video:** 15–30 seconds, hook in first 2 seconds, captions always on
- **Branding:** Use brand colors (#541409, #EAD6D6, #F7F3ED) in overlays
- **Typography:** Playfair Display for headlines in graphics
- **Photography:** Real product photos only — never stock images
- **Text on image:** Keep under 20% of image area (Meta best practice)

## Landing Page Strategy

### Principle: Match Ad Intent to Landing Page

| Ad Type              | Landing Page       | Why                                    |
|----------------------|--------------------|----------------------------------------|
| General brand        | `/menu`            | Let them browse all products           |
| Cookie-specific      | `/cookies`         | Direct to the product they searched    |
| Cake-specific        | `/cakes`           | Direct to cakes with CTA to order      |
| Wedding              | `/weddings`        | Wedding-focused messaging and gallery  |
| Holiday/seasonal     | `/collection/[season]` | Seasonal landing page              |
| Retargeting          | `/order`           | Order hub — they already know the brand|

### Landing Page Optimization

- Hero image should match the ad creative (visual continuity)
- CTA visible without scrolling on mobile
- Price visible within first viewport
- "100% Gluten-Free" trust badge prominent
- Page load under 3 seconds (Core Web Vitals)
- No distracting navigation for ad-specific landing pages (consider simplified header)

## UTM Convention

Use consistent UTM parameters for all paid traffic:

```
utm_source = [platform]       → facebook, instagram, google, pinterest
utm_medium = [ad type]        → cpc, cpm, social, display
utm_campaign = [campaign name] → spring-launch-2026, valentines-cookies, wedding-awareness
utm_content = [ad variant]     → carousel-v1, video-15s, static-hero
utm_term = [keyword]           → gf-bakery-cincinnati (Google only)
```

### Example URLs

```
bakesbycoral.com/cookies?utm_source=facebook&utm_medium=cpc&utm_campaign=spring-launch-2026&utm_content=carousel-v1
bakesbycoral.com/cakes?utm_source=google&utm_medium=cpc&utm_campaign=gf-cakes-local&utm_term=gluten-free-cake-cincinnati
bakesbycoral.com/weddings?utm_source=pinterest&utm_medium=cpm&utm_campaign=wedding-awareness&utm_content=pin-vertical-v1
```

## Ad Performance Benchmarks

### Expected Metrics (Small Local Business)

| Metric              | Facebook/IG     | Google Search  | Pinterest      |
|---------------------|-----------------|----------------|----------------|
| CTR                 | 1.5–3.0%       | 3–6%           | 0.5–1.5%       |
| CPC                 | $0.50–$1.50    | $1.00–$3.00    | $0.30–$0.80    |
| Conversion Rate     | 2–5%           | 5–10%          | 1–3%           |
| Cost per Order      | $10–$30        | $10–$25        | $15–$40        |

### When to Scale

- Scale a campaign when cost per order is under $15 (50% of $30 cookie order)
- Pause campaigns with cost per order over $25
- Test new creative every 2–4 weeks to prevent ad fatigue
- Seasonal campaigns should start 3–4 weeks before the holiday
