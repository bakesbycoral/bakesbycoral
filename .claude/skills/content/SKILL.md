## SEO Writing Guidelines

### On-Page SEO Checklist

- [ ] **Title tag:** Include primary keyword, under 60 characters
- [ ] **Meta description:** Include keyword + CTA, 150–160 characters
- [ ] **H1:** One per page, includes primary keyword
- [ ] **H2/H3:** Use secondary keywords in subheadings
- [ ] **First 100 words:** Include primary keyword naturally
- [ ] **Internal links:** Link to 2–3 relevant product/order pages
- [ ] **Image alt text:** Descriptive, includes keywords naturally
- [ ] **URL slug:** Short, keyword-rich, hyphenated (e.g., `/blog/gluten-free-wedding-cake-cincinnati`)

### Keyword Integration Rules

- Write for humans first, search engines second
- Use primary keyword 3–5 times naturally in a 1,000-word post
- Use variations and synonyms (gluten-free, GF, celiac-safe, celiac-friendly)
- Include location naturally (Cincinnati, Ohio, Greater Cincinnati)
- Never keyword-stuff — if it sounds forced, rewrite it

### Content Quality Standards

- Every post should provide genuine value — not just exist for SEO
- Include personal perspective (Coralyn's voice, "I" statements)
- Use real product photos, not stock images
- Include at least one CTA linking to a product or order page
- Proofread for accuracy on pricing, lead times, and product details

## Repurposing Framework

Every blog post should generate:
1. **2–3 social media posts** (key excerpts, quotes, images)
2. **1 newsletter mention** (summary + link)
3. **Pinterest pin(s)** (vertical image + keyword-rich description)
4. **Instagram carousel** (key tips or points from the post)
